Arithmetic Operators

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • **Exponetiation Ex. 2**3 == 2*2*2
  • % Modulus (Division Remainder)
  • ++ Increment
  • -- Decrement

Assignment Operator

  • = Assignment

Comparison Operators

  • == Equal to
  • === equal to and equal type
  • != not equal
  • !== not equal value and not equal type
  • > greater than
  • < less than
  • >= greater than or equal to
  • <= less than or equal to
  • ? ternary operator Ex. 4==5 ? "true" : "false"
    • if(4==5){ //true }else{ // false }

Logical Operators

  • && Logical and
    • (2>1 && 5<10)
  • || Logical or
    • (2>1 || 5>10)
  • ! Logical not
    • (!true)
