jQuery Height and Width Method

Width Methods
  • width() // Return Content width
  • innerWidth() // Return Content width + padding (top + bottom)
  • outerWidth() // Return Content width + padding (top + bottom) + Border width (top + bottom)
  • outerWidth(true) // Return Content width + padding (top + bottom) + Border width (top + bottom) + margin (top + bottom)
    • Height Methods
  • height() // Return Content height
  • innerHeight() // Return Content Height + padding (left + right)
  • outerHeight() // Return Content Height + padding (left + right) + Border Height (left + right)
  • outerHeight(true) // Return Content Height + padding (left + right) + Border Height (left + right) + margin (left + right)
